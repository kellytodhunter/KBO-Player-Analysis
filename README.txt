# KBO Player Analytics Project

## Overview
Built a performance evaluation system for KBO hitters and pitchers using advanced statistics and standardized metrics.

## Methods
- Feature engineering (HR/PA, K/9, WHIP, etc.)
- Z-score normalization
- Composite performance indices (HPI & PPI)
- Weighted performance models

## Outputs
- Player rankings for hitters and pitchers
- Correlation analysis of pitching metrics
- Visual comparisons of model performance

## Tech Stack
Python, pandas, numpy, matplotlib, seaborn

## Key Insights
- WHIP strongly correlates with ERA
- Strikeout rate adds independent value
- Weighted indices improve ranking stability
